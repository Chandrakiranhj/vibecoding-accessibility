When building or editing web UI, preserve accessibility:

- Use native HTML controls before custom ARIA widgets.
- Give every form control a visible connected label or explicit accessible name.
- Give every icon-only button/link an accessible name and hide decorative icons.
- Keep keyboard focus visible and logical.
- Announce loading, validation, async responses, route changes, and toasts.
- Check contrast at the token/component level.
- Give charts text summaries and data-table fallbacks.
- Respect reduced motion and use a tested modal/dialog primitive.
